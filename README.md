1. 非法雇佣未成年人：该个人在我 14 岁时明知故犯地与我建立了口头工作关系。经过数月的合作发展，他单方面终止了我们的合作，并在我的社交和职业圈内散布诽谤性的谣言。
2. 恶意的法律威胁与勒索：该个人试图通过捏造严重且毫无根据的刑事指控（如“信用卡欺诈”）来反悔之前的自愿性财务赠予（明确不是贷款）。他的明确意图是将民事问题刑事化，利用司法程序作为勒索的手段。
3. 利用脆弱性进行剥削：利用年龄、社会经验和权力动态方面的巨大差异，该个人进行了长期的心理压力施加。他威胁要使用不正当的行政手段来扰乱我们的运营，这是对未成年人合法权利的明显侵犯。非法持有及泄露个人数据威胁：他声称非法占有并威胁要泄露属于我和我家人的重要个人信息（SPI）。根据中国严格的个人信息保护法（PIPL），此类行为构成严重违规，可能会因侵犯公民个人信息而面临刑事指控。
4. 此次开源发布是对这些行为的直接回应。这是为了保护我们的权利，并对抗我们所遭遇的不道德行为所采取的必要措施。

## Abstract

AlphaGPT 是一套基于深度学习算法与符号回归的量化因子挖掘框架。通过构建非线性、高阶的特征组合，在 A 股回测中表现出极强的鲁棒性与超额收益捕捉能力。
